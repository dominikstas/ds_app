Projekty

MusicX

Github

Aplikacja internetowa wykonana przy użyciu Django, SQLite, HTML i CSS. Strona pozwala rejestrować kolekcje płyt, wykorzystując bazę danych oraz formularze. W fazie rozwoju znajduje się opcja dodawania zamówień pre-orderowych, które w dniu wydania albumu, automatycznie dodawałaby się do kolekcji.

NoteX

Github

Prosty, lekki edytor tekstu, utworzony wykorzystując Pythona, PyQT oraz SQLite. Zapewnia podstawowe funkcje, takie jak zapis i otwieranie plików, tryb ciemny, dodawanie nagłówków czy tworzenie list.
